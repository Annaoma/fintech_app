# Xend Finance App

## Created by
Chris Anna (chris Elizabeth)

## Project Description
This is a fintech mobile application built using React Native and Expo.

The app includes:
- Welcome Screen
- Login Screen
- Email Authentication
- Home Dashboard
- Referral Screen
- Wallet and Plans Navigation

The purpose of this app is to simulate a digital banking and crypto-finance platform.

## Technologies Used
- React Native
- Expo Router
- TypeScript
- VS Code

## How to Run
1. Install dependencies using npm install
2. Run the app using npx expo start
